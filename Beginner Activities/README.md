# BEAPER Pico Beginner Activities

These beginner programming activities are designed to introduce novices to
microcontroller programming using MicroPython running on the Raspberry Pi Pico
family of microcontrollers. These activities are created specifically for use
with the BEAPER Pico circuit hardware -- feel free to modify them for use with
other hardware devices.

## Curriculum Overview

- the **Microcontroller Core Curriculum.md** file provides a complete curriculum outline
